# 第4章 字符串的扩展

# 新增方法

ES6中字符串这块关于编码方面做了一些扩展，但是实际开发中用的可能并不多，有兴趣的同学可以做扩展学习一下。

1、includes：返回布尔值，表示是否找到了参数字符串；

'abc'.includes('a');    // true

2、startsWith：返回布尔值，表示参数字符串是否在源字符串的头部；

'abc'.startWith('b');   // false

3、endsWith：返回布尔值，表示参数字符串是否在源字符串的尾部；

'abc'.endsWith('b');   // false

4、repeat：返回一个新字符串，便是将原字符串重复n次；如果n是小数，则向下取整，负数或者Infinity则会报错。其中0 ~ -1之间的小数取整为0，NaN也等同于0，如果是字符串也会被转为数字，不能转为有效数字就为0。

'x'.repeat(3);  // "xxx"

5、模板字符串 这个使用场景很多，尽量使用它取代传统的+号拼接字符串；

`abc${参数名}`

模板字符串中还能调用函数

`abc,${fn()}, bar`
